# Count Vowels in a String (JavaScript)

## Description
A simple JavaScript challenge to **count all vowels** (a, e, i, o, u) in a given string.  
This project is designed for beginners to practice **string manipulation**, **loops**, and **conditional statements** in JavaScript.

## Challenge
Write a function `countVowels(text)` that takes a **string** as input and returns the **number of vowels** in it.  
Vowels are: a, e, i, o, u (case-insensitive).

### Requirements
- Function should accept any string as input.
- Treat uppercase and lowercase vowels equally (`A = a`).
- No external libraries; use plain JavaScript.

### Examples

```js
countVowels("Hello World") // => 3
countVowels("JavaScript")  // => 3
countVowels("OpenAI")      // => 3
